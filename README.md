
# Run and deploy your AI Studio app

This contains everything you need to run your app locally.


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## How to Start
1. Visit my.telegram.org to generate your **API ID** and **API Hash**.
2. Enter your credentials in the **Parameters** sidebar. Ensure the phone number includes your country code.
3. Navigate to the **Export Code** tab once your settings are validated.
4. Download the extractor.py file and run it in your local Python terminal.

## Technical Features
1. CSV Exporting
Comprehensive member lists including User IDs, Usernames, and First/Last names.
2. vCard Packaging
Automatically generates .vcf files compatible with mobile contact imports (phones only).
3. Flood Protection
Built-in delay mechanisms and exponential backoff for FloodWait errors.
